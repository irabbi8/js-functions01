# Problem-Solving Brief

- Use the exact function name
- Declare all variables inside the function
- Use console log inside the function
- If function is called, make it validate all parameter queries
- Use one JS file for all functions
- Call all the functions for different inputs

### Regex Used

- `/.../` encloses the regex
- `^` declares the start of string
- `$` declares the end of string
- `+` means one or more times
- `*` means zero or more times
- `[...]` means any character from the set
- `[a-zA-Z]` means either lowercase or uppercase letters
- `[ .'-]` means either space, dot, apostrophe or hyphen
